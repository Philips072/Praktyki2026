Projekt składa się z dwóch części:

| Folder | Technologia | Adres |
|--------|-------------|-------|
| `projekt/DataMindAi/` | React + Vite + Supabase | `http://localhost:5173` |
| `projekt/backend/` | Node.js + Express | `http://localhost:3001` |
